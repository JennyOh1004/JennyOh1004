Yelp Clone

Developed a Yelp Clone that integrates with the Google Map API and includes features user comments, star ratings, image uploading, and user authentification.

The Web Application is to support user entered places, which indicates to store the details about the places users have entered someplace. Also, the Web Application supports user authentication, that is the ability for a user to exist and be able to Log in and Log out.

Created Web Application using the Ruby, Ruby on Rails framework, HTML, CSS and Bootstrap.
