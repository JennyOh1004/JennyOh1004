Flixter Yoga

Created the Flixter Yoga web application using HTML, CSS, JS, Bootstrap and Ruby on Rails framework since I have a huge passion for creating the yoga web appliction.

Developed a Video Streaming marketplace platform, featuring credit card payment capabilities, user role management, complex User Interfaces and advanced database relationship.

This web application contains many categories such as instructor pages, student pages, enrollment page, sign in and sign out pages. When I built this project, I had to fix many bugs such as pending problems, database problems and so on. When I found the errors for the first time, I started researching it over and over until I found the solution. Also I tried to figure it out until I fix this problem. For example, when I had a pending error, I needed to drop and recreate the local database. Also I needed to restart the heroku and database too. I experienced the same issue several times. If I encounter this problem again, It can be fixed easily with my knowledge and experience. I realized that I have learned a lot of helpful things from this project because now I know how to solve those problems. For this reason, I believe that It was a very rewarding project.
